TÍTULO DO PROJETO: Decodificador de Binário
VERSÃO ou DATA: 16/10/2022
OBJETIVO DO PROJETO: Criar um programa em Java que faça a decodificação de números em bináros.
COMO INICIAR O PROJETO: Inicie a classe no método main e siga as instruções do programa.   
OBJETIVO DO PROJETO: O programa solicita um número em binário e a quantidade de bits e retorna o número inteiro sem sinal, um número inteiro em complemento de 2 e se houver 8 bits ou menos, também retorna o caracter representado pela sequência em binário.
Após a impressão dos dados na tela, o usuário pode escolher continuar verificando outros números em binários ou parar o programa.
